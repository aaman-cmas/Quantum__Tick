Introduction

Overview of the Digital Stopwatch Project

A Digital Stopwatch is a precise timing device used to measure elapsed time in seconds. This project was designed using Digital Logic Design principles without any microcontroller or programming. The stopwatch uses 7490 counters, 7447 decoders, 7408 AND gates, and 7-segment displays.


The entire circuit was simulated using NI Multisim and displays time from 00 to 59 seconds in real-time. The project helps students understand BCD counting, decoding, sequential logic circuits, and digital display systems.

Project Objectives

Main goals of the stopwatch system

Design Stopwatch

Create a 2-digit digital stopwatch using Digital Logic Design concepts.

Use Counters

Implement modulo-10 and modulo-6 counting using 7490N ICs.

Display Timing

Connect decoders with 7-segment displays to show real-time counting.

Simulation

Test and observe the stopwatch functionality using NI Multisim.

Components Used

Hardware and digital logic components

7490N IC

  • BCD decade counter
  • Counts from 0 to 9
  • Provides binary outputs

7447N IC

  • BCD to 7-segment decoder
  • Controls display segments
  • Supports common anode displays

7408N AND Gate

  • Contains four AND gates
  • Used for control and reset logic

7-Segment Display

  • Displays numeric digits
  • Used for real-time stopwatch output

Signal Generator

  • Produces 1Hz clock pulse
  • Acts as stopwatch ticking signal

Power Supply

  • Provides +5V DC
  • Powers all digital ICs

Working Principle

How the digital stopwatch functions

Clock Pulse Generation

The signal generator creates 1 pulse per second which acts as the timing signal.

Counting Process

The first 7490 counter counts from 0–9 while the second counts from 0–5.

BCD Conversion

The counter outputs are sent to 7447 decoders which convert BCD into display signals.

Display Output

The 7-segment displays show the elapsed time from 00 to 59 seconds.

Conclusion

Project Summary

The Digital Stopwatch project was successfully implemented using digital logic components. The system accurately counted time from 00 to 59 seconds using 7490 counters and displayed the results through 7-segment displays controlled by 7447 decoders. This project enhanced understanding of sequential circuits, BCD conversion, digital counters, and real-time display systems.
