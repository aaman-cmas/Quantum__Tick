Introduction

Overview of the Digital Stopwatch Project

A Digital Stopwatch is a precise timing device used to measure elapsed time in seconds. This project was designed using Digital Logic Design principles without any microcontroller or programming.


The stopwatch uses 7490 counters, 7447 decoders, 7408 AND gates, and 7-segment displays.


The entire circuit was simulated using NI Multisim and displays time from 00 to 59 seconds in real-time.

Project Objectives

Main goals of the stopwatch system

Design Stopwatch

Create a 2-digit digital stopwatch using DLD concepts.

Use Counters

Implement modulo-10 and modulo-6 counting using 7490N ICs.

Display Timing

Connect decoders with 7-segment displays.

Simulation

Test and observe the stopwatch functionality in NI Multisim.

Components Used

Hardware and digital logic components

7490N IC

  • BCD decade counter
  • Counts from 0 to 9
  • Provides binary outputs

7447N IC

  • BCD to 7-segment decoder
  • Controls display segments
  • Supports common anode displays

7408N AND Gate

  • Contains four AND gates
  • Used for reset/control logic

7-Segment Display

  • Displays numeric digits
  • Shows stopwatch output

Signal Generator

  • Produces 1Hz clock pulse
  • Acts as stopwatch ticking signal

Power Supply

  • Provides +5V DC
  • Powers all digital ICs

Working Principle

How the digital stopwatch functions

Clock Pulse Generation

The signal generator creates 1 pulse per second.

Counting Process

The first counter counts 0–9 while the second counts 0–5.

BCD Conversion

7447 decoders convert BCD into display signals.

Display Output

7-segment displays show the elapsed time.
